# On-Device AI Lab – Overview

Este repo agrupa diferentes ejemplos de ejecución de AI on-device:

- `miyabi-nano`: Gemini Nano en Android (AICore).
- `iki-nano`: Gemma 2B en iOS usando MediaPipe Task GenAI.

## Objetivo

Explorar diferentes runtimes y plataformas para ejecutar modelos pequeños on-device:
- Gemini Nano vía AI Core.
- Gemma 2B vía MediaPipe.
- Futuras pruebas con otros runtimes (ONNX, TFLite, etc.).
